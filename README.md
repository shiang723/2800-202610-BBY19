# Vancooler
Vancooler is a dynamic web application that helps pedestrians, outdoor workers, and commuters find real-time shade while navigating the city. Unlike static maps, our app will provide a shadow-aware experience, visualizing where shadows fall across sidewalks and public spaces based on the current time of day and the city’s building heights.
